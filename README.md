# Gestão de Clientes e Empresas com Módulos (RI4)

Este projeto é uma atividade acadêmica desenvolvida para o 3° semestre da **FATEC São José dos Campos**, focada no aprofundamento de Programação Orientada a Objetos (POO) utilizando **JavaScript** moderno.

## Sobre o Projeto

O sistema simula um cadastro de clientes com gestão de endereços e telefones através de **composição de objetos**. O objetivo principal é exercitar o uso de atributos privados, métodos de acesso (getters/setters) e algoritmos de ordenação.

### Tecnologias Utilizadas
* **Linguagem:** JavaScript (ES6+)
* **Conceitos:** Encapsulamento Real (`#`), Composição, Imutabilidade e Ordenação Alfabética.
* **Ambiente:** Desenvolvido e testado utilizando **VS Code** e **Node.js**.

## Funcionalidades

* **Encapsulamento Estrito:** Uso de campos privados para garantir a integridade dos dados.
* **Formatação de Dados:** Métodos integrados para recuperação de atributos em caixa alta e caixa baixa.
* **Saída Formatada:** Exibição detalhada no console organizando informações de cliente, telefone e endereço.

## Como Executar

1.  **Clone este repositório:**
    ```bash
    git clone https://github.com/jopaul0/RI4.git
    ```
2.  **Importe o projeto:**
    Abra a pasta do projeto no VS Code.
3.  **Execute o script de teste:**
    Certifique-se de ter o Node.js instalado e execute o arquivo principal:
    ```bash
    node index.js
    ```
